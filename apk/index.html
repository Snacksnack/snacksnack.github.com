Installation

A simple installation manual.

1. First scroll down to the downloads, and then download preferably the latest version.
2. Use any file manager, or get one from the store if you need and open the downloads folder.
It's in the ~/sdcard/download by default. 3. Open the apk to start the installer.

3a. It might tell you the phone doesn't allow unknown apk's.
You will have to allow those first then.
If it gives you an 'settings' option then use that, otherwise go to Settings > Applications 3b. Check the following box. 4. After this the installer will work and ask you for the usual permissions.
Accept these and you're done.

Usage

A simple usage manual

If you open the application you will see this. Since you probably never used it go to settings, enter your technician name and save that. After this you can open the photo menu. Click on 'new' to create a new task to upload.
Enter your order number and then take, or pick pictures.
Optionally check delete files to delete them.
Added a wrong file? Hold the item and a popup will appear.
Want to see a preview of the file? Click it. When you've picked a file it will ask you what type is it.
Pick a type and it'll be added to the list. After clicking 'send files' a new window will open and that will send the files. If the upload fails due to whatever reason it will be added to the queue.
Want to remove a task? Hold the item and a popup will appear. And if it succeeds to the history.
Which can be cleared with a simple button push.
